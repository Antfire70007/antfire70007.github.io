    CloudAtCost 雲端主機服務使用心得

    Published:    February 9, 2023
    Category:    心得感想
    Written by:    Antfire
    Views:    
    Comments:    
    Likes:   0

    CloudAtCost是一間雲端機器服務供應商，在很久很久以前，有購買他宣稱花一次錢，終身使用的服務。我購買的是的開發伺服器資源，大概花了50多美金。fdsf

    使用一段時間後，我發現他們會用各種名義向你收取費用，例如機器維護費等。

    以我購買的資源來說，維護費是 9刀
    我覺得使用者付費，日常的機器維護也是需要人力，到這邊我還可以接受，但讓我不舒服的是你一開始也沒提到到時候會有這些費用的產生，只有一直強調花一次錢終身使用這件事。

    另一件令人不舒服的是，他們公司有一陣子一直推出新的服務(CloudPRO v3)，並要求你要升級否則機器可能會怎麼樣怎麼樣的，升級也是要付錢，要付多少錢就看你手上有的資源有哪些，當然資源越多升級所需費用也越高。

    我看到那個升級金額實在有點吃不消，於是就選擇不升級，就這樣擺著，反正還可以用。

    然後接著又推出CloudPRO的升級版：RS Cloud Services； 而正式機的服務變成Dedicated Servers

    直到2022? 我發現他們開始推銷新的項目－雲端挖礦，如今已找不到相關網頁了，但我還有信件留底。

    直到有一天，我發現我的部落格打不開了，登入CloudAtCost後台，發現我的機器被關機，然後我的開發伺服器的收費居然無預警改成月費制了，我必須要先繳費，才能把機器開起來繼續用，也沒有收到任何的信件通知。

    於是我當下立馬把我的部落格搬家到Github上，並且想辦法把我的網域也一起想辦法設定到Github Page。

    有想尋找便宜的雲端機器的朋友，良心建議千萬不要考慮CloudAtCost，這間黑心公司，請考慮GCP、Azure或是Amazon等有品質保證的服務。

    向擅自巧立名目收費及變更服務的黑心公司說不！
