    Hexo 基本操作

    Published:    November 3, 2021
    Category:    Information
    Written by:    Antfire
    Views:    
    Comments:    
    Likes:   0

     快速啟動站台

    輸入以下指令，快速開啟站台看看成果
    hexo server

    開啟完成後，進入http://localhost:4000/

     撰寫文章

    輸入以下指令撰寫一篇文章

    hexo new post hello-world

    hello-world 將會是你的Markdown檔案名稱，會產生在 source/_posts 資料夾底下

     撰寫草稿

    輸入以下指令撰寫文章草稿
    hexo new draft hello-world
    hello-world 將會是你的Markdown檔案名稱，會產生在 source/_drafts 資料夾底下

     放置靜態檔案

    如果有想要放圖片、pdf、txt等檔案的話，可以將檔案放置在 source資料夾底下，Hexo會將source底下的檔案複製一份到站台的根目錄，例如我放了一份test.txt到source資料夾底下，我想要存取的話，以本機為例，網址就是http://localhost:4000/test.txt 。

     溫馨提醒

    不用擔心_posts或是_drafts的檔案是不是也會一併被複製過去，系統會忽略以_開頭的名稱，如果檔案或資料夾是以_開頭的話，將忽略該檔案或資料夾而不會複製過去。

     產生靜態檔案

    輸入以下指令將產生靜態檔案
    hexo generate或hexo g

     參考資料

    https://hexo.io/zh-tw/docs/writing
